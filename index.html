Part 1

什麼是 CC0？

Creative Commons Zero — 公眾領域貢獻宣告

  • ✓ 免費使用 — 不用付費
  • ✓ 免標註來源 — 不用寫「圖片來源：XXX」
  • ✓ 可商業使用 — 做 YouTube 影片完全沒問題
  • ✓ 可自由修改 — 裁切、調色、加文字都行
從 Google 隨便找的圖片不是 CC0！使用有版權的素材可能被 YouTube 下架。

Part 2

Suno 是什麼？

  • 🎵 一個 AI 音樂生成工具，輸入文字就能產出完整歌曲
  • 🎸 支援各種曲風：流行、搖滾、電子、古典、中文歌...
  • 🆓 免費帳號每天有一定額度可使用

👉 前往 suno.com 註冊 / 登入

可以用 Google 帳號快速登入！
Part 2

基本操作：兩大欄位

🎨 Style Prompt（風格描述）

告訴 AI 你想要什麼風格

Mandarin pop, female vocal, emotional piano ballad, slow tempo

📝 Lyrics（歌詞）

填入歌詞，搭配結構標籤

[Verse] 月光灑落在窗前 思念隨風飄遠 [Chorus] 我在這裡等你回來
Part 2

[ ] 中括號 — 歌曲結構標籤

用 [ ] 標記段落結構，幫助 AI 組織音樂

標籤用途說明
[Intro]前奏歌曲開頭，營造氣氛
[Verse]主歌主要敘事段落
[Pre-Chorus]導歌情緒漸進，銜接副歌
[Chorus]副歌最高潮、最有記憶點的部分
[Bridge]橋段轉折段，帶來變化
[Outro]尾奏歌曲收尾
Part 2

[ ] 更多實用標籤

標籤效果
[Instrumental]純樂器演奏，沒有人聲
[Break]間奏 / 節奏暫歇
[Solo]樂器獨奏
[Rap]饒舌段落
[Breakdown]降低能量，拉回樂器
[Build]能量逐漸堆疊
標籤可以組合，例如 [Verse 2]、[Final Chorus]
Part 2

[ ] 完整歌詞範例

[Intro] [Instrumental] [Verse 1] 城市的燈光太刺眼 找不到回家的路線 [Chorus] 但你是我的北極星 在黑暗中指引方向 [Break] [Verse 2] 雨滴落在肩膀上 像你溫柔的手掌 [Chorus] 但你是我的北極星 在黑暗中指引方向 [Outro]
Part 2

( ) 小括號 — 聲音效果指令

用 ( ) 控制演唱方式和聲音表現

語法效果
(whispered)低語、氣音
(belted)高亢嘶吼
(spoken word)口白，說話而不是唱
(harmonized)和聲效果
(falsetto)假音
(ad-lib)即興發揮
Part 2

( ) 進階用法

🎤 二重唱 / 回音效果

歌詞後加括號重複，產生合唱感：

我在這裡等你 (我在這裡等你) 永遠不會離開 (永遠不會離開)

🎵 插入氣氛短語

[Chorus] (oh yeah!) 今晚我們一起飛 飛到星空的那一端 (hey!)
Part 2

( ) 控制情緒變化

括號描述段落的情緒轉變：

[Verse 1] (softly, gentle) 那天你走的時候 天空下著小雨 [Chorus] (building intensity) 但我知道你會回來 因為我們的故事還沒結束 [Bridge] (stripped back, piano only) 如果時間能倒流...
括號指令放在段落開頭效果最好！
Part 2

歌詞修正三種方式

AI 生成的歌不滿意？有三種方法調整：

1. Rerun

同樣歌詞，重新生成
→ 旋律和編曲會完全不同

2. Extend

從歌曲結尾繼續延長
→ 適合歌曲太短或被截斷

3. 手動修改

複製歌詞修改後重新 Create
→ 最精確的控制方式

Part 2

手動修改歌詞（詳細步驟）

  • 聽完生成的歌曲，找到不滿意的地方
  • 看右側欄位的歌詞，找到對應段落
  • 複製歌詞到左側 Lyrics 欄位
  • 修改想改的部分（錯字、段落、標籤）
  • 其他欄位（Style 等）保持不變
  • 點擊 Create 重新生成
如果只是在某處斷掉，複製斷點之後的歌詞用 Extend 更方便！

Part 3

Vizzy.io
影片製作

把音樂和圖片變成一支炫酷的音樂影片

Part 3

Vizzy.io 是什麼？

  • 🎬 線上 音樂視覺化影片 製作工具
  • 🎵 圖片可以隨著音樂節拍跳動
  • 📊 內建音頻分析器 Audio Analyzer
  • ✏️ 支援加入動態文字
  • 🆓 免費，瀏覽器直接操作
  • 🎞️ 可匯出最高 4K / 60fps

👉 vizzy.io → 點擊 Editor

建議使用 Chrome 瀏覽器，相容性最好！
Part 3

步驟 1 — 建立專案 & 匯入音樂

  • 到 vizzy.io 點擊 Editor 進入編輯器
  • 刪除預設的範例內容（選取 → Delete）
  • 將 Suno 下載的 MP3 檔案拖曳進編輯器
  • 確認音樂出現在時間軸上
進入編輯器後，點選上方 Help 可以看官方教學！
Part 3

步驟 2 — 設定背景圖片

  • 將 Part 1 下載的 CC0 圖片拖曳進編輯器
  • 調整圖片大小，填滿整個畫面
  • 可使用多張圖片，在時間軸上排列切換
想讓背景隨音樂跳動？加上 Audio Analyzer！
Part 3

步驟 3 — 加入音頻視覺化

  • 找到 Audio Analyzer（音頻分析器）
  • 加到想要的視覺元素上
  • 調整 Base 和 Max 數值控制反應強度
  • 播放預覽效果，不滿意就微調

可以加視覺化的元素：

  • 背景圖片的縮放 / 旋轉
  • 音波波形 Sound Wave
  • 文字的大小 / 透明度
Part 3

步驟 4 — 加入動態文字

  • 加入 Text 文字元素，輸入歌名
  • 選擇字體、大小、顏色
  • 可加歌詞字幕，設定出現時間
  • 搭配 Audio Analyzer 讓文字跟節拍跳動
需要中文字體？請使用 CC0 / OFL 授權的字體！（回顧 Part 1）
Part 3

步驟 5 — 預覽 & 匯出

  • 按 播放 預覽整支影片
  • 不滿意隨時調整
  • 點擊 Export / Render
  • 建議解析度：1920 × 1080（1080p）
  • 等待匯出完成，下載 MP4
匯出需要時間，趁這時候去準備 YouTube 的標題和描述！
Part 3

Part 4

上傳步驟

  • 前往 studio.youtube.com（登入 Google）
  • 點擊右上角 建立 → 上傳影片
  • 選擇匯出的 MP4 檔案
  • 填寫標題和描述
標題：你的歌名 | AI Music Video 描述： 🎵 音樂：Suno AI 生成 🎬 影片：Vizzy.io 製作 📸 圖片：Unsplash / Pexels (CC0)
  • 可見度：公開 或 不公開（只有連結的人能看）
  • 點擊 發布 🎉
Part 4

上傳前檢查清單

  • ✓ 圖片素材都來自 CC0 圖庫？
  • ✓ 字體使用的是免費授權字體？
  • ✓ 音樂是用 Suno 自己生成的？
  • ✓ 影片解析度至少 1080p？
  • ✓ 影片有聲音、畫面正常？
全部用 CC0 素材 + AI 生成的音樂 = 不會有版權問題！
