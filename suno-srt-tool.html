Suno SRT 字幕下載工具

一鍵從 Suno 歌曲頁面匯出帶時間軸的 .srt 字幕檔

方法一：書籤列按鈕（推薦）

把下方按鈕拖曳到你的 Chrome 書籤列，就完成安裝了！

下載 Suno SRT 字幕

把這個按鈕拖到書籤列即可

如果看不到書籤列？請先按 Ctrl+Shift+B（Windows）或 Cmd+Shift+B（Mac）顯示書籤列。

Mac 版教學

Windows + Chrome 安裝步驟

  1. 開啟 Chrome 瀏覽器
  2. 按 Ctrl+Shift+B 顯示書籤列（如果還沒顯示的話）
  3. 用滑鼠左鍵按住上方紅色的「下載 Suno SRT 字幕」按鈕
  4. 拖曳到書籤列上，放開滑鼠
  5. 書籤列上會出現「下載 Suno SRT 字幕」— 安裝完成！

使用方式

  1. 到 suno.com 登入你的帳號
  2. 打開你想下載字幕的歌曲頁面（網址會像 suno.com/song/xxxxx）
  3. 點擊書籤列上的「下載 Suno SRT 字幕」
  4. 等幾秒後會自動下載 .srt 檔案
下載的 .srt 檔案可以直接拖進 Vizzy、CapCut、Premiere 等影片編輯工具當字幕使用。

Mac + Chrome 安裝步驟

  1. 開啟 Chrome 瀏覽器

  4. 拖曳到書籤列上，放開
  5. 書籤列上會出現「下載 Suno SRT 字幕」— 安裝完成！

常見問題

Q: 點了沒反應？

請確認你已經登入 Suno，而且目前頁面是某首歌的頁面（網址包含 /song/）。

Q: 顯示「此歌曲沒有字幕資料」？

表示這首歌還沒有生成對齊歌詞。通常純演奏曲（Instrumental）不會有字幕。有歌詞的歌曲需要在生成完成後稍等片刻才有字幕資料。

Q: .srt 檔案可以做什麼？

SRT 是標準字幕格式，可以用在：

  • Vizzy.io — 匯入當字幕
  • CapCut（剪映）— 匯入字幕軌
  • YouTube — 上傳影片時手動上傳字幕
  • Premiere Pro / DaVinci Resolve — 專業剪輯

工具基於 Suno 公開 API，僅供學習用途
